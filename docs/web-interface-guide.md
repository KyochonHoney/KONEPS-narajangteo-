## 🔍 입찰공고 조회 및 관리

### 📊 통계 대시보드
공고 목록 페이지 상단에 4개 카드로 표시:
- **전체 공고**: 수집된 총 공고 수
- **활성 공고**: 아직 마감되지 않은 공고
- **오늘 수집**: 오늘 새로 수집된 공고
- **마지막 업데이트**: 최근 수집 시간

### 🔎 검색 및 필터링
- **검색어**: 제목, 기관명, 공고번호로 검색
- **상태**: 활성/마감 공고 필터
- **분류**: 용역/공사/물품 분류별 필터
- **기간**: 시작일~종료일 범위 설정

### 📋 공고 목록 기능
각 공고마다 다음 액션 버튼 제공:
- **👁️ 상세**: 공고 상세 정보 페이지
- **🔗 나라장터**: 원본 나라장터 페이지로 새 창에서 이동
- **🗑️ 삭제**: 공고 데이터 삭제

### 📄 공고 상세 페이지
- **기본 정보**: 공고번호, 제목, 내용, 발주기관 등
- **계약 정보**: 예산, 계약 방법, 자격 요건 등
- **일정 정보**: 공고일, 마감일, 개찰일 등
- **원본 보기**: 나라장터 원본 페이지 직접 링크

## 🚀 사용 시나리오

### 📋 시나리오 1: 일반적인 공고 조회
1. `https://nara.tideflo.work` 접속
2. 관리자 계정(`admin@nara.com` / `admin123`)으로 로그인
3. 상단 네비게이션에서 "공고 목록" 클릭
4. 원하는 공고의 "상세" 버튼 클릭
5. "나라장터" 버튼으로 원본 페이지 방문

### 🔍 시나리오 2: 고급 필터링으로 정밀 수집
1. 로그인 후 "공고 목록" → "데이터 수집" 클릭
2. "고급 필터링" 라디오 버튼 선택
3. 수집 기간 설정 (예: 최근 30일)
4. 원하는 지역 선택 (예: 서울, 경기)
5. 업종 코드 선택 (예: 1426, 1468)
6. 원하는 인증서 코드 선택
7. "데이터 수집 시작" 클릭
8. 완료 후 "수집된 공고 보기"로 결과 확인

### 📈 시나리오 3: 수집 데이터 관리
1. 공고 목록에서 필요없는 공고 삭제 (휴지통 아이콘)
2. 정기적으로 "데이터 수집" 실행하여 최신 공고 확보
3. 검색 기능으로 관심 공고 필터링
4. 각 공고의 나라장터 링크로 원본 정보 확인

## 💡 팁 & 주의사항

### ✅ 사용 팁
- **고급 필터링**: 사용자가 원하는 정확한 조건으로 공고만 선별 수집 가능
- **나라장터 링크**: 모든 공고에서 원본 페이지로 바로 이동 가능
- **실시간 진행률**: 수집 과정을 실시간으로 모니터링
- **검색 활용**: 공고번호로 검색하면 특정 공고를 빠르게 찾을 수 있음
- **필터 조합**: 여러 필터를 조합해서 원하는 공고만 정밀 검색 가능

### ⚠️ 주의사항
- **로그인 필수**: 모든 기능은 로그인 후 사용 가능
- **관리자 권한**: 일부 기능은 관리자 계정만 접근 가능
- **API 연결**: 나라장터 API 상태에 따라 수집 결과가 달라질 수 있음
- **대용량 수집**: 긴 기간의 데이터 수집 시 시간이 오래 걸릴 수 있음
- **브라우저 권장**: 최신 Chrome, Firefox, Safari 권장
