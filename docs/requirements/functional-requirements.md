# 기능 요구사항 명세서

## 1. 사용자 인증 및 권한 관리

- **FR-002**: 로그인 실패 시 5회 초과 시 계정 일시 잠금 (30분)
- **FR-003**: 비밀번호 찾기 기능 제공 (이메일 인증)
- **FR-004**: 로그인 상태 유지 옵션 제공 (Remember Me)

### 1.2 권한별 접근 제어
